## Jogo de Adivinhação
Este é um projeto de jogo de adivinhação de números.

# Tecnologias
HTML
CSS
JS

# Sobre o projeto
Este este projeto é um jogo de adivinhação de números aleatórios gerados pelo sistema como tentativas limitadas

# Objetivo
O objetivo do projeto é treinar e testar habilidades adquiridas em HTML, CSS e JavaScript, bem como utilização de lógica de programação, funções e versionamento via Git.

# Próximos passos:
O próximo passo planejado é a criação do modo difícil, onde o range de números aumenta para 1 - 200, a quantidade de tentativas diminui para 7 e apenas a dica mais recente fica visível
